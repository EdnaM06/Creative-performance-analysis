# 🎨 Creative Performance & Digital Marketing Analytics

## 🧠 Project Overview

This project analyzes digital marketing campaign data, ad events, and user session behaviour to understand how different creatives and campaigns perform across an eCommerce funnel.

The goal is to simulate a real-world growth analytics problem:

> How do we identify high-performing ads and optimise marketing spend to improve conversions and ROI?

---

## 🏢 Business Context

Digital marketing platforms run thousands of campaigns simultaneously, generating large volumes of event-level data.

Companies like **Ruby Labs** rely on this type of data to:
- Optimize ad spend efficiency
- Improve campaign targeting
- Identify high-performing creatives
- Increase conversion rates across the funnel

However, challenges include:
- Too many campaigns to evaluate manually
- Lack of clarity on which creatives actually drive conversions
- Difficulty connecting ad engagement to revenue outcomes

---

## 📊 Dataset Overview

Source: Demo Data (Digital Marketing Ads & Events Sample)

### Dataset Structure:
- 1,000 ad campaigns
- 10B events per year (simulated scale)
- Multiple relational tables

### Tables included:
- `dim_campaigns` → campaign metadata
- `dim_keywords` → keyword targeting data
- `fact_ad_events` → ad-level interactions
- `fact_campaign_ads` → campaign-ad relationships
- `fact_session_events` → user session behaviour

---

## 🎯 Objective

To analyze campaign, ad, and user event data in order to:

- Identify top-performing campaigns and creatives  
- Understand user engagement across sessions  
- Evaluate campaign effectiveness across channels  
- Detect patterns in high-converting ads  
- Provide data-driven recommendations for optimization  

---

## 💡 Business Goal

Improve marketing ROI by:
- Scaling high-performing campaigns  
- Reducing spend on low-performing ads  
- Improving targeting strategy using engagement data  
- Connecting ad engagement to user behavior and conversions  

---

## 🧰 Tools Used

- Python (Pandas, NumPy)
- SQL (data querying & joins)
- Matplotlib / Seaborn (visualization)
- Jupyter Notebook
- Power BI (future dashboard)

---

## 🧭 Analysis Approach

1. Data exploration across campaign tables  
2. Join ad events with campaign metadata  
3. Analyze engagement patterns (clicks, impressions, sessions)  
4. Identify high vs low performing creatives  
5. Extract business insights and recommendations  

---

## 📈 Expected Insights

- Which campaigns generate the highest engagement  
- Which creatives drive the most user interaction  
- Which keywords improve conversion performance  
- Where marketing budget is being wasted  
- How user sessions correlate with ad performance  

---

## 🚀 Business Impact

If applied in a real company environment:
- Improved ad ROI through better targeting  
- Reduced wasted ad spend  
- Higher conversion rates from optimized creatives  
- Better understanding of customer journey from ad → purchase
